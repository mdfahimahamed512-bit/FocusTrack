# FocusTrack

FocusTrack is a minimal study time tracking web application designed to measure focused study duration and break patterns.

## Features
- Real-time study timer
- Break tracking
- Focus time calculation
- Clean UI
- Zero backend

## Tech Stack
- HTML
- CSS
- JavaScript

## Deployment
1. Upload to GitHub
2. Import into Vercel
3. Deploy

## Use Case
Helps students analyze study habits and focus efficiency.
